## Core tags
| Tag Name        | Meaning / Use                                                 | Attributes       | 
| --------------- | ------------------------------------------------------------- | ---------------- | 
| `<h1>` – `<h6>` | Headings — `<h1>` is the largest, `<h6>` the smallest.        |
