# Decoration
## font from Google
```html
<!-- get the font weight 300 and 900 -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;900&display=swap" rel="stylesheet">
```

## font-size
Change all font-size properties from px to rem, letting the user to change the default page sizing by zooming in and zooming out. 
1rem = 16px
```css
p {
      font-size: 1rem;
}
```

## underline
```css
a {
	text-decoration:underline dotted;
}
```

## Links and buttons
- links are for navigation, go to another location.  
- Buttons are for interaction. (Open a model)  
You can style the link like a button.
